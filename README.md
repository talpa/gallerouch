# Gallerouch

Moderní galerie a e-shop pro malířku

- Frontend: React, Redux, i18n, axios
- Backend: Node.js/Express
- Databáze: PostgreSQL
- Docker Compose pro orchestraci
- Uživatelské účty, správa objednávek, platební brána, generování PDF faktur

## Struktura
- `/frontend` – React aplikace (galerie, e-shop, správa účtu)
- `/backend` – Node.js/Express API (uživatelé, objednávky, platby, faktury)
- `/docker-compose.yml` – orchestrace služeb

## Vývoj
- Lokální i Docker vývoj
- Připraveno pro rozšíření o další funkce
